# msrs

This is a Rust project initialized with Cargo.

## Getting Started

To build the project:

```
cargo build
```

To run the project:

```
cargo run
```

## Project Structure
- `src/main.rs`: Main source file
- `Cargo.toml`: Project manifest

## Requirements
- [Rust](https://www.rust-lang.org/tools/install)
